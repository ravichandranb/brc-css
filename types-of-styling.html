CSS stands for Cascading Style Sheet.
That means styling the elements of HTML document.
Styling for HTML document can be given in 3 ways.

1. External CSS
2. Internal CSS
3. In-Line CSS
EXTERNAL CSS
Styling is defined in an external CSS file.
We use this file using LINK element.

INTERNAL CSS
Style for a particular HTML is defined within the
HTML document itself within the head section.
example:
h1{color: red;}
In this example, the header characters in h1
will be colored with red fonts.
IN-LINE CSS
Styling is defined within the HTML tag itself.
"style=colour:red" as attribute
within a tag.
